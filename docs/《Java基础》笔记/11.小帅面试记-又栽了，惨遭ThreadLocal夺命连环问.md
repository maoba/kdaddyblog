这些问题，如同阴影一般，在小帅的脑海里挥之不去。

是的，他万万没想到，自诩“多线程小能手”的他栽在了ThreadLocal上。

这是小帅苦投了半个月简历之后才拿到的面试机会，然而又丧失了。当下行情实在是卷到了极点。

都两个月了，面试机会少，居然还每次都被问翻，这样下去真要回老家种地了，小帅内心五味成杂......
